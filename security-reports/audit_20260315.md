## Identified Issues

1. **Missing Rate Limiting**: Several public GET endpoints lack rate limiting, facilitating DoS attacks.
2. **Weak Validation**: Role updates use generic `dict` instead of strict Pydantic models.
3. **Pydantic Deprecations**: Using old V1 config style.
